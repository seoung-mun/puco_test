# imgs

제품 UI와 데모에서 재사용하는 수동 관리 이미지 자산 루트입니다.

## 하위 문서

- [building/README.md](building/README.md)
- [parts/README.md](parts/README.md)
- [plantation/README.md](plantation/README.md)
- [role/README.md](role/README.md)

## 현재 구조

- `overview.png`: 전체 보드/배치 참고 이미지
- `parts/`: 자원, 승점, 식민자, 배 등 공용 파츠
- `plantation/`: 농장/채석장 타일
- `building/`: 현재 비어 있는 건물 전용 자산 자리
- `role/`: 현재 비어 있는 역할 카드 전용 자산 자리

## 메모

- 프론트 컴포넌트가 직접 import하는 소형 자산보다, 보드게임 도메인 자산 저장소에 가깝습니다.
